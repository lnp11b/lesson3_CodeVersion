Highlight #1 Repository

What is a repository? A repository is a place for a programmer to store their code on. It is a place where multiple files are located for distribution over a network. Using a GitHub repository allows the user to work on any computer.

Highlight #2 Git

Git is a VCS that allows a user to save data and make changes to it overtime.Every time you commit, Git takes a picture of what your files look like at that specific moment and stores a references to that snapshot. If nothing has changed to the file Git will not store the file again. Git, compared to other VCS, is more like a mini filesystem.

Highlight #3 VCS

VCS or version control system is a system that records changes to a file over time. This will allow the user to recall specific versions later allowing you to open view any time in the future.It provides what, when, and where you did the changes in your file.
